Hello there!

I'm Michael, a web developer/designer, game developer, programmer, and in general, a technology enthusiast. I create websites, games, programs, and other things in the realms of technology. Want to learn more? Check out the about page.

On GitHub you can find the source code for many of the open source things I'm working on (most of it being code for developers). On the projects page you'll find pages for some of them (the ones I think are worth showing). Check it out and see what you can find.
