Uncorrupted is the game I created in less the 48 hours for Ludum Dare 22, in which it scored place #34 out of around 720 entries (here's the top 50). It's a platformer/shooter kind of game, where you're a robot (who's remarkably human-like in thinking) with no idea where or who you are. It's apparent that you're in some kind of research facility. It's also apparent that all the researchers are dead, lying on the ground. But why?

When playing, make sure to read all the notes, logs, and boards, as most of them either contain information as to what happened, or real-world references.

Source and License

You can find the full source for the game at its GitHub repository. Be warned that because of the tight time limit, the code is more messy organisational-wise than I would've liked. Everything (code, art, sound, etc.) is licensed under the zlib/libpng license.

Software Used

  • FlashPunk - The awesome Flash game engine.
  • TextMate - Text editor to write the code.
  • Pixen - Pixel art to make the art.
  • Bfxr - Sound effect generator.
  • Audacity - Sound editor which I used to convert all sound effects to MP3 format.
  • Adobe Photoshop Elements - Cut down version of the most used image editor, Photoshop. I used this (version 6) to create the light that appears at the end.
